# VIN Buyer Confidence Prototype

Simple static prototype for the BMW Buy/Pass Review landing page.

## Run locally

From this folder:

```bash
python3 -m http.server 8787
```

Then open:

- http://127.0.0.1:8787

## Files

- `index.html` - landing page prototype
